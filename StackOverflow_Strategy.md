## 📝 内容策略

### 1. 问题解答策略：

#### 高质量解答标准：
- **完整性**：全面解答问题，覆盖所有方面
- **准确性**：技术内容准确无误
- **实用性**：提供可实施的解决方案
- **结构化**：清晰的结构和格式
- **代码示例**：提供完整可运行的代码示例

#### 解答模板：
```
## 问题分析
[简要分析问题核心]

## 解决方案
### 方案1：[主要方案]
- 优点：[优点说明]
- 缺点：[缺点说明]
- 适用场景：[适用情况]

### 方案2：[备选方案]（如有）
- 优点：[优点说明]
- 缺点：[缺点说明]
- 适用场景：[适用情况]

## 代码示例
```python
# 完整可运行的代码示例
def solution():
    # 实现代码
    pass
```

## 性能分析
- 时间复杂度：O(...)
- 空间复杂度：O(...)
- 优化建议：[优化建议]

## 相关资源
- [相关文档链接]
- [进一步学习资源]
- [工具推荐]

[简要介绍相关服务，非硬性推广]
```

### 2. 问题选择策略：

#### 优先回答的问题类型：
1. **高浏览量问题**：浏览量高，影响力大
2. **未解决难题**：长时间未解决的技术难题
3. **代码审查问题**：直接相关服务的问题
4. **架构设计问题**：系统设计相关问题
5. **算法优化问题**：性能优化相关问题

#### 避免回答的问题类型：
1. **过于简单问题**：基础问题，价值低
2. **重复问题**：已有高质量解答的问题
3. **模糊问题**：需求不明确的问题
4. **非技术问题**：与技术服务无关的问题
