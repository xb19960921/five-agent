## 🚨 识别问题（8类问题）

### 🔴 高严重度问题（3个）

#### 1. 类型检查缺陷
```python
elif isinstance(item, int):  # ⚠️ bool 是 int 的子类！
    item = str(item)
```
**问题**：`bool`是`int`的子类，`isinstance(True, int)`返回`True`，导致布尔值被错误转换为字符串`"True"`/`"False"`
**风险**：数据语义丢失，可能引发下游逻辑错误

#### 2. 重复项多次计数
```python
if item in seen:
    duplicates.append(item)  # ⚠️ 每次遇到重复都添加，导致重复项多次出现
```
**问题**：输入`[1, 1, 1]`输出`[1, 1]`，而非预期的`[1]`
**风险**：数据统计错误，违反函数语义

#### 3. 嵌套列表爆炸
```python
# 多次合并同一键会创建深层嵌套：[value1, value2, value3] → [[value1, value2], value3]
merged[key] = [merged[key], value]
```
**问题**：多次调用合并同一键会产生嵌套列表，数据结构不可预测
**风险**：数据一致性破坏，难以调试

### 🟡 中严重度问题（3个）

#### 4. Pythonic反模式
```python
for i in range(len(data_list)):  # ⚠️ 应该直接迭代
    item = data_list[i]
```
**问题**：使用索引迭代是Python反模式，降低可读性且稍慢

#### 5. 使用字典而非集合
```python
seen = {}  # ⚠️ 应该用 set()
seen[item] = 1
```
**问题**：字典比集合占用更多内存，且语义不清晰

#### 6. 缺少类型提示与文档字符串
**问题**：所有函数均无类型注解和docstring，降低可维护性

### 🟢 低严重度问题（2个）

#### 7. 异常处理缺失
**问题**：未处理输入类型错误（如传入非迭代对象）

#### 8. 魔法数字
```python

```

## 🛠️ 重构后代码

```python
from typing import Any, List, Dict, Union, Optional
from collections import defaultdict

def process_data(
    data_list: List[Any], 
    float_precision: int = 2,
    skip_none: bool = True
) -> List[Union[str, int, float]]:
    """
    处理混合类型数据列表，进行类型标准化。
    
    Args:
        data_list: 输入数据列表
        float_precision: 浮点数保留精度，默认 2 位
        skip_none: 是否跳过 None 值，默认 True
    
    Returns:
        标准化后的数据列表
        
    Examples:
        >>> process_data([" hello ", 42, 3.14159, None])
        ['hello', '42', 3.14, None]
    """
    result = []
    for item in data_list:
        if item is None:
            if skip_none:
                continue
            result.append(None)
            continue
        
        # ⚠️ 先检查 bool（bool 是 int 的子类）
        if isinstance(item, bool):
            result.append(item)
        elif isinstance(item, str):
            result.append(item.strip())
        elif isinstance(item, int):
            result.append(str(item))
        elif isinstance(item, float):
            result.append(round(item, float_precision))
        else:
            continue
    
    return result


def find_duplicates(items: List[Any]) -> List[Any]:
    """
    查找列表中的重复项（每个重复项只返回一次）。
    
    Args:
        items: 输入列表
        
    Returns:
        重复项列表（去重）
        
    Examples:
        >>> find_duplicates([1, 2, 2, 3, 3, 3])
        [2, 3]
    """
    seen = set()
    duplicates = set()
    
    for item in items:
        if item in seen:
            duplicates.add(item)  # ✅ 使用集合避免重复
        else:
            seen.add(item)
    
    return list(duplicates)


def merge_dicts(
    dict1: Dict[str, Any], 
    dict2: Dict[str, Any],
    merge_strategy: str = "list"
) -> Dict[str, Any]:
    """
    合并两个字典，冲突键按策略处理。
    
    Args:
        dict1: 基础字典
        dict2: 待合并字典
        merge_strategy: 冲突处理策略
            - "list": 将冲突值收集到列表（默认）
            - "override": dict2 覆盖 dict1
            - "keep_first": 保留 dict1 的值
    
    Returns:
        合并后的字典
        
    Examples:
        >>> merge_dicts({"a": 1}, {"a": 2})
        {'a': [1, 2]}
        >>> merge_dicts({"a": 1}, {"a": 2}, "override")
        {'a': 2}
    """
    merged = dict1.copy()
    
    for key, value in dict2.items():
        if key in merged:
            if merge_strategy == "override":
                merged[key] = value
            elif merge_strategy == "keep_first":
                continue
            elif merge_strategy == "list":
                if isinstance(merged[key], list):
                    merged[key].append(value)
                else:
                    merged[key] = [merged[key], value]
        else:
            merged[key] = value
    
    return merged
```
